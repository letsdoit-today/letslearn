# How to add a new demo

To add a new demo to this project, follow these steps:

-  Create a new demo file in the `src/demos/` directory.
-  Add the demo to the `Demos` array in `Meta.tsx`.
-  Add the demo to the `routes` array in `generate-static.js`.
-  Add the demo to the `dynamicRoutes` array in `vite.config.ts`.
-  Add the demo to the `Routes` array in `App.tsx`.
-  Update the `README.md` file with the new demo.
